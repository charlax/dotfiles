# Vim

## Tags

Source: https://vim.fandom.com/wiki/Browsing_programs_with_tags

Check tags configuration in `ctags/ctags`.

- Go to tag: `Ctrl-]`
- Preview a tag: `Ctrl-W }`
- Return after a tag jump: `Ctrl-[`, `:pop`
