# Web tools

One-page, vibe-coded HTML files.

Run with:

```bash
python3 -m http.server
```

Then open: http://localhost:8000/

## Inspiration

- [Useful patterns for building HTML tools](https://simonwillison.net/2025/Dec/10/html-tools/), Simon Willison
- [simonw/tools: Assorted useful tools, almost entirely generated using LLMs](https://github.com/simonw/tools?tab=readme-ov-file)
