Prerequisites
=============

Only tested on Mac Os X 10.6. Homebrew should be installed.

Installation
============

    $ cd ~
    $ git clone git@github.com:charlax/dotfiles.git .dotfiles
    $ python .dotfiles/install.py

You may have to remove some pre-existing files manually.
